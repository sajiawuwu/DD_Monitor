# DD监控室

## 运行指南

确保安装VLC, `HELP.html`内有更多解释。

## 开发指南

推荐在venv/anaconda环境下开发

### 所需包
 - PyQt5

 - requests

 - aiowebsocket

 - python-vlc

 - pyinstaller

 - dnspython    

   

   pip安装

   ```bash
   pip install -r requirements.txt
   ```

   

### 打包

在 `scripts` 文件夹下有各平台的打包脚本，需要在仓库根目录运行。
